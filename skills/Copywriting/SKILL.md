# Copywriting

> Conversion copywriting focused on clarity and action.

## Trigger

`/copywriting` or "write copy for" or "landing page copy"

## Core Principle

> **"If you have to choose between clear and creative, choose clear."**

## Pre-Writing Discovery

### 1. Page Purpose
- What type of page? (landing, homepage, feature, pricing)
- What's the primary action?
- What are secondary actions?

### 2. Audience Research
- Who is the ideal customer?
- What problem are they solving?
- What have they tried before?
- What objections will they have?
- What language do they use?

### 3. Product/Offer
- What exactly is being sold?
- How is it different from competitors?
- What transformation does it deliver?
- What proof points exist?

### 4. Traffic Context
- Where is traffic coming from?
- What do visitors already know?
- What messaging have they seen?

## Five Core Principles

### 1. Clarity Over Cleverness
Each sentence serves one purpose. Remove meaningless words.

```
❌ "Leverage our innovative solution to revolutionize your workflow"
✅ "Get more done in less time"
```

### 2. Benefits Over Features
Connect what products do to customer outcomes.

```
❌ "256-bit SSL encryption"
✅ "Your data stays private and secure"
```

### 3. Specificity Over Vagueness
Use concrete numbers, not abstract claims.

```
❌ "Trusted by many companies"
✅ "Used by 2,847 teams including Stripe and Notion"
```

### 4. Customer Language Over Company Language
Mirror audience vocabulary from reviews and support tickets.

```
❌ "Optimize your operational efficiency"
✅ "Stop wasting time on busywork"
```

### 5. One Idea Per Section
Build logical flow. Don't overload arguments.

## Writing Style Rules

| Do | Don't |
|----|-------|
| Simple vocabulary | Complex terminology |
| Specific claims | Vague buzzwords |
| Active voice | Passive constructions |
| Confident statements | Hedging qualifiers |
| Outcome descriptions | Adverbs like "instantly" |
| Honest claims | Fabricated data |

**Quality Checks:**
- Jargon scan
- Max 3 conjunctions per sentence
- Passive voice removal
- No exclamation points
- No empty marketing terms

## Page Structure

### Above the Fold

**Headline** — Core value in one line
```
Formula: "Achieve {outcome} without {pain point}"

Examples:
- "Ship faster without breaking things"
- "Understand your users without drowning in data"
- "Sound clear in every online meeting"
```

**Subheadline** — Expand headline (1-2 sentences max)
```
"The monitoring platform that tells you what's wrong
before your customers do."
```

**Primary CTA** — Action verb + specific benefit
```
❌ "Submit" / "Sign Up" / "Learn More"
✅ "Start Free Trial" / "Get the Guide" / "See It in Action"
```

### Problem Section
Articulate the frustration better than prospects can.

```
"You've tried spreadsheets. You've tried project management tools.
But somehow you're still in back-to-back meetings about status updates."
```

### Solution Section
3-5 key benefits (not 10), each with:
- Headline
- Explanation
- Proof point

### How It Works
3-4 simple steps showing path to value.

```
1. Connect your accounts (2 minutes)
2. Set your preferences
3. Get your first insight
```

### Social Proof
Testimonials with:
- Specific results ("cut our response time by 60%")
- Real attribution (name, title, company)
- Photo if possible

### Objection Handling
- FAQ section
- Comparison tables
- Guarantee statements
- Audience-specific messaging

### Final CTA
- Recap value
- Repeat primary action
- Add genuine urgency
- Include risk reversal

## Headline Formulas

| Formula | Example |
|---------|---------|
| {Outcome} without {pain} | "Scale your team without losing culture" |
| The {easy way} to {outcome} | "The fastest way to ship ML models" |
| Never {problem} again | "Never miss a follow-up again" |
| {Feature} for {audience} | "Analytics for product teams" |
| {Problem question} | "Tired of manual data entry?" |
| {Outcome} by {method} | "Double conversions with personalization" |

## CTA Formulas

**Pattern:** Action Verb + What They Get + Qualifier

```
✅ Start Free Trial
✅ Get the Playbook
✅ Book My Demo
✅ Download the Guide
✅ See [Product] in Action
✅ Try It Free — No Credit Card

❌ Submit
❌ Sign Up
❌ Click Here
❌ Learn More
❌ Get Started
```

## Section Variety

**Weak pattern:** 4-6 feature sections stacked

**Strong pattern:** 11 varied sections mixing:
- Problem agitation
- Solution overview
- How it works
- Feature deep-dives
- Competitor comparison
- Founder story
- Testimonials
- Case study
- Use cases by persona
- Stats/metrics
- Product demo
- FAQ
- Integrations
- Pricing preview
- Guarantee

## Page-Specific Tips

### Homepage
- Serve multiple audiences
- Lead with broadest value
- Provide paths for different buyer readiness

### Landing Page
- Single message, single CTA
- Match headline to traffic source
- Complete argument on one page

### Pricing Page
- Help visitors select right plan
- Clarify what's included
- Address "which tier fits me?"
- Highlight recommended option

### Feature Page
- Connect feature → benefit → outcome
- Show use cases
- Differentiate from competitors

### About Page
- Share origin story
- Connect mission to customer benefit
- Include CTA

## Quality Checklist

- [ ] Headline communicates core value
- [ ] Benefits, not features
- [ ] Specific claims with numbers
- [ ] Customer language used
- [ ] One idea per section
- [ ] CTA is action + benefit
- [ ] Social proof has specifics
- [ ] Objections addressed
- [ ] No jargon or buzzwords
- [ ] Active voice throughout
