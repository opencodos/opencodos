# Telegram MCP Integration Plan

## Overview

Set up [telegram-mcp](https://github.com/chigwell/telegram-mcp) as an MCP server for Claude Code, enabling direct Telegram messaging from Atlas.

## Architecture

```
Claude Code → MCP Protocol → telegram-mcp → Telegram API
                                ↓
                        Reuse existing session.string
```

## Priority Tools (Tier 1)

These 10 tools cover 90% of DKOS use cases:

| # | Tool | Use Case | DKOS Workflow |
|---|------|----------|---------------|
| 1 | `send_message` | Send messages | `msg Pat "hi"` |
| 2 | `get_messages` | Read recent messages | Morning brief inbox |
| 3 | `search_messages` | Find content in chats | Research, context lookup |
| 4 | `list_chats` | List all conversations | Inbox overview |
| 5 | `get_chat` | Get chat details | Check unread, metadata |
| 6 | `reply_to_message` | Reply in context | Follow-up messages |
| 7 | `resolve_username` | Name → chat ID | Find contacts by name |
| 8 | `mark_as_read` | Mark processed | After processing inbox |
| 9 | `search_contacts` | Find people | CRM lookup |
| 10 | `get_last_interaction` | Last message with person | Relationship tracking |

## Implementation Steps

### Step 1: Clone telegram-mcp

```bash
cd $CODOS_PATH/ingestion/
git clone https://github.com/chigwell/telegram-mcp.git
cd telegram-mcp
```

### Step 2: Install dependencies

```bash
# Using uv (recommended)
uv sync

# Or pip
pip install -r requirements.txt
```

### Step 3: Configure credentials

Create `.env` file in telegram-mcp directory:

```bash
# Link to existing secrets
ln -s $CODOS_PATH/dev/Ops/.env .env
```

Or create new `.env`:

```env
TELEGRAM_API_ID=<from Dev/Ops/.env>
TELEGRAM_API_HASH=<from Dev/Ops/.env>
TELEGRAM_SESSION_STRING=<from existing session.string>
```

### Step 4: Copy existing session

```bash
# Reuse authenticated session from existing agent
cp ../Telegram-agent/session.string ./
```

### Step 5: Configure Claude Code MCP

**This is auto-generated by bootstrap.sh** in `~/.claude/.mcp.json`.

If you need to manually configure, add to `~/.claude/.mcp.json`:

```json
{
  "mcpServers": {
    "telegram": {
      "command": "uv",
      "args": [
        "--directory",
        "$CODOS_PATH/ingestion/telegram-mcp",
        "run",
        "main.py"
      ]
    }
  }
}
```

Replace `$CODOS_PATH` with your actual path (e.g., `/Users/yourname/projects/codos`).

### Step 6: Test connection

```bash
# Verify session works
cd $CODOS_PATH/ingestion/telegram-mcp
uv run main.py
```

### Step 7: Restart Claude Code

```bash
# Restart to load MCP server
claude --mcp-debug  # Shows MCP tool loading
```
